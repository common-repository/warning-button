== Description ==

This plugin add a "warning button" at the bottom of your content to send notification to the administrator.
when you click the button, an email notification is sent to the administrator in order to indicate that the current page contains inappropriate content.
You can set this informations :
* Email of the administrator
* Subject of the notification
* A template for the email body
This extension is compatible with multisite.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all files to the `/wp-content/plugins/warning-button` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the administration to set all parameters
